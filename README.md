# The Conversion Leak Audit

A premium landing page for @samthecopypro's conversion leak audit service.

## Overview

This is a **static HTML landing page** optimized for GitHub Pages hosting. It features:

- **Responsive Design**: Mobile-first, works on all devices
- **Dark Premium Aesthetic**: Deep navy background (#0a0a0f) with electric red accents (#ff2d2d)
- **MailerLite Integration**: Email capture form that sends submissions directly to MailerLite
- **Fast Loading**: Pure HTML, CSS, and vanilla JavaScript—no frameworks needed
- **SEO Optimized**: Meta tags and semantic HTML structure

## Files

- `index.html` - Main landing page
- `styles.css` - All styling (responsive design included)
- `script.js` - MailerLite form integration and smooth scroll

## Deployment

This site is hosted on GitHub Pages. Any changes pushed to the `main` branch will automatically deploy.

### To deploy:
1. Push changes to GitHub
2. GitHub Pages automatically builds and deploys
3. Site is live at `https://yourusername.github.io/conversion_leak_audit`

## Email Capture

Email submissions are handled by **MailerLite** (form ID: 186512759014295403). When visitors submit their email:

1. The form is submitted to MailerLite's servers
2. MailerLite handles the email validation and storage
3. A success message is displayed to the visitor
4. You receive the submission in your MailerLite account

## Design System

**Typography:**
- Headlines: Bebas Neue (letter-spacing: 0.52px)
- Body: DM Sans (400 weight)

**Colors:**
- Background: #0a0a0f (deep navy)
- Text: #ffffff (white)
- Accent: #ff2d2d (electric red)
- Secondary: #1a1a2e (muted dark grey)

## Customization

To modify the landing page:

1. Edit `index.html` for content changes
2. Edit `styles.css` for design/layout changes
3. Edit `script.js` for functionality changes
4. Push to GitHub to deploy automatically
